项目结构

10 个教学项目，覆盖数据库运维全生命周期

01

了解数据库运维工作

认识数据管理发展历史、数据库类型与结构，熟悉数据库运维常见工作内容，了解 DBA 职位职责及行业发展趋势。

数据管理发展 关系型/非关系型 DBA职责 智能运维趋势
02

安装和配置 MySQL 数据库

在 Windows 和 Linux 系统下完成 MySQL 的下载、安装、配置与启动，掌握 MySQL 服务管理、目录结构及基本工具使用。

MySQL 8.0 安装 配置文件 服务管理 mysql命令行
03

管理 MySQL 权限与安全

创建和管理 MySQL 用户账户，授予与收回权限，配置远程访问，实施数据权限、结构权限与管理权限的精细控制。

用户创建 GRANT / REVOKE 远程访问 权限刷新
04

分析 MySQL 日志

掌握错误日志、一般查询日志、慢查询日志和二进制日志的配置与查看方法，使用 mysqldumpslow 分析慢查询，使用 mysqlbinlog 实现基于时间点和位置的精确数据恢复。

错误日志 慢查询日志 二进制日志 mysqlbinlog
05

备份与恢复 MySQL 数据库

使用 mysqldump 实现逻辑备份，使用 Percona XtraBackup 实现热备份，基于 crontab 配置自动备份作业，完成跨版本数据库迁移及数据表导入导出。

mysqldump XtraBackup crontab 数据迁移
06

监控、测试并优化 MySQL 性能

使用 MONyog 监控服务器状态，使用 sysbench 和 mysqlslap 进行压力测试，使用 explain 分析执行计划并优化 SQL 语句，调优关键配置参数。

MONyog sysbench explain 参数调优
07

MySQL 复制

在 Windows 和 Linux 下搭建 MySQL 主从复制，配置 GTID 复制、半同步复制与并行复制，实现读写分离架构，提升数据库可用性与性能。

主从复制 GTID 半同步复制 并行复制
08

搭建及运维 MySQL Cluster

认识 MySQL Cluster 节点架构与运行原理，在 Windows 和 Linux 下搭建 MySQL Cluster，实现节点的启动、停止、联机备份与数据恢复。

Cluster架构 节点管理 联机备份 日志管理
09

结合 Redis 的 MySQL 运维

在 Windows 和 Linux 下安装配置 Redis，掌握 Redis 数据结构与读写原理、常用命令及配置项，搭建 Redis Cluster，实现 MySQL+Redis 读写分离。

Redis安装 Redis命令 Cluster集群 读写分离
10

数据库自动化运维

了解数据库运维从人工→工具→产品的演进历程，掌握自动化运维平台架构与功能，使用 goInception 完成 SQL 审核，配置智能巡检与告警，实现自动闭环运维。

goInception 智能巡检 自动化告警 运维平台
